# Quickstart: Lab Diagram Tests

## Testing the Feature

This quickstart guide describes how to test the implementation of the Lab Diagram Tests feature.

### Prerequisites

-   The application is running locally.
-   You are logged in as an instructor.

### Steps

1.  **Navigate to the Lab Creation Page**: Go to the page for creating a new lab.
2.  **Create a New Lab**: Give the lab a title and start adding pages.
3.  **Add a Question Page**:
    -   Add a new page.
    -   Select the "Question" type.
    -   Choose "Multiple Choice" and fill in the prompt, options, and select the correct answer.
4.  **Add a Diagram Test Page**:
    -   Add another new page.
    -   Select the "Diagram Test" type by enabling the toggle.
    -   Verify that the diagram editor is displayed with a palette of shapes.
    -   Create a simple diagram.
5.  **Save as Draft**:
    -   Navigate between the pages.
    -   Verify that a "draft saved" notification appears.
6.  **Publish the Lab**:
    -   Go to the final page of the lab creation process.
    -   Click the "Publish" button.
    -   Verify that the lab is now published.
7.  **View as Student**:
    -   Log in as a student.
    -   Find the newly published lab.
    -   Verify that you can view the content of the lab, including the question and the diagram.
